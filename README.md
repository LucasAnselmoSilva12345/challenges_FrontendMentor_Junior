## Visão geral

### O desafio

Este desafio se trata da criação de uma aplicação estilo dashboard, informando os dados de views, acesso às postagens e likes e entre outros filtros, dentre as principais redes sociais: Facebook, Instagram, Twitter e YouTube. 

Os valores passados desses dados, são valores estáticos, e a aplicação também conta com a opção de troca de tema.
